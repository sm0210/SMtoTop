##实例化插件方法：

<b>//实例化SMtoTop 一</b>

$().initSMtoTop({smToTopHide:false});

<b>//实例化SMtoTop 二</b>

$().initSMtoTop({smToTopTxt:'Top',smToTop:10,smTopTopRight:300,smToTopBottom:50});

<b>//实例化SMtoTop 三</b>

$().initSMtoTop({smToTopTxt:'toTop',smTopTopRight:450,smToTopBottom:35});
